# Education 

## Final Project 
This is my final project and it was a very good experience for me because I build this program from scratch, and the idea of the program it is new in Saudi Arabia, 

## What's the need?
the parents face issues when looking for a good school for their children, like the need to go to each school and ask the school if the status of registration is available or not, and they have a lot of questions about the school.

## Proposed solution
Based on Saudi 2030 Vision with digital transformation, I have developed the Schools app, because there is no program that has the same idea in Saudi Arabia, and the idea is which allows users to know the schools and the user can see the comment for the school and the details for each school 

## Why Education?
- **Arabic:** The app targeted Saudi Arabia, so it was designed using the Arabic language.

## App main screens
As above, my app contains the following screens:
- animation 
- main page (display the schools)
- Signup / Login / (app authenticate users using email and password)
- Reset Password (app sends an email to the user with reset password link)
- Profile (contains user personal information)

## App main features
- User can use the program without sign 
- User can Signup / Login / Reset Password
- User can view the profile
